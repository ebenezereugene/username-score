# username-score
An open-source, deterministic username scoring engine for TypeScript. 



if youve ever signed up to any social media like x(formerly twitter) or instagram and your name was taken, and gibberish was assigned to you, this package is to fix that....so instead of you being assigned gibberish like for instance jimoh and you get jimoh23415 jimoh8900 you get more human names like iamjimoh,jimohmedia, jimohprime, j1moh....the task was simple, how may we design a package that takes a user name and tells how good it is and also suggest better names that could fill in for that.
